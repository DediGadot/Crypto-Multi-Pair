9. Portfolio Rebalancer

Portfolio

Multi-asset portfolio with systematic threshold-based rebalancing. Research-backed approach that outperforms buy-and-hold by forcing "buy low, sell high" discipline.

Strategy Logic
Maintains target asset allocations and rebalances when weights drift beyond threshold. Supports three rebalancing methods:
  • Threshold: Rebalance when deviation > threshold (e.g., 15%)
  • Calendar: Rebalance on fixed schedule (e.g., monthly)
  • Hybrid: Rebalance when either condition is met
Parameters
Parameter Default Description
assets - List of (symbol, weight) tuples
rebalance_threshold 0.15 15% deviation triggers rebalance
rebalance_method threshold threshold/calendar/hybrid
min_rebalance_interval_hours 24 Minimum hours between rebalances
calendar_period_days 30 Days for calendar rebalancing
use_momentum_filter false Skip rebalancing in strong trends
Rebalancing Logic
REBALANCE: When asset weight deviates > threshold from target
Action: Sell overweight assets, buy underweight assets
Result: Systematically "sell high, buy low"
